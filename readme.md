## 📖 Project Overview

**Pearl Fast Food** is a modern, fully responsive restaurant website built with pure HTML5, CSS3, and JavaScript. The site features an interactive food menu, shopping cart functionality, customer reviews, and a sleek user interface designed to provide an exceptional dining experience online.

**🔗 Live Demo:** [https://daniyal-khan-pearl-fast-food.netlify.app/](https://daniyal-khan-pearl-fast-food.netlify.app/)

## ⚡ Quick Start

```bash
# Clone the repository
git clone https://github.com/daniyal-khan-dev/pearl-fast-food.git

# Navigate to project directory
cd pearl-fast-food
```

## 🛠️ Tech Stack

### Core Technologies
- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with flexbox, grid, and animations
- **JavaScript (ES6+)** - Interactive functionality and DOM manipulation

### External Libraries
- **Swiper.js** - Touch-enabled carousel/slider functionality
- **SweetAlert2** - Beautiful, responsive popup notifications
- **Font Awesome** - Comprehensive icon library
- **EmailJS** - Contact form email integration

## ✨ Key Features

### 🎨 Design & UX
- **Fully Responsive Design** - Seamless experience across all devices
- **Modern UI/UX** - Clean, intuitive interface with smooth animations
- **Custom Loading Animation** - Branded loader with professional feel
- **Smooth Scrolling Navigation** - Single-page application experience

### 🍽️ Restaurant Features
- **Interactive Food Menu** - 15+ food categories with detailed views
- **Shopping Cart System** - Add, remove, and manage orders
- **Wishlist Functionality** - Save favorite dishes for later
- **Image Carousels** - Multiple images per dish with navigation
- **Customer Reviews** - Testimonials with star ratings

### 🔧 Technical Features
- **LocalStorage Persistence** - Cart and wishlist data saved locally
- **Modal Windows** - Detailed dish views and checkout process
- **Form Validation** - Real-time input validation for contact/checkout
- **Email Integration** - Contact form with EmailJS
- **Performance Optimized** - Fast loading with efficient asset management

## 🏗️ Architecture

### Static Architecture Benefits
- **Performance** - Fast loading with CDN delivery
- **Scalability** - Can handle high traffic without server load
- **Security** - No server-side vulnerabilities
- **Cost-effective** - Minimal hosting requirements

### Data Storage Strategy
- **localStorage** - Cart and wishlist persistence
- **Session Storage** - Temporary form data
- **No Database** - Purely client-side application
